Facade

Proporcionar una interfaz simplificada a un subsistema complejo de clases.

Estructural

📖 Descripción

Facade define una interfaz de alto nivel que hace al subsistema más fácil de usar. Oculta la complejidad interna de múltiples clases y sus interacciones, ofreciendo métodos simples que orquestan las operaciones necesarias. Los clientes usan la fachada en lugar de interactuar directamente con el subsistema.

🎯 Casos de uso prácticos

1. Sistema de home theater

Una sola llamada `verPelicula()` coordina encender proyector, bajar persianas, configurar audio y reproducir.

2. API de autenticación

Una fachada orquesta validación de credenciales, generación de token JWT, logging y auditoría.

💻 Ejemplos de código

JavaScript
// Subsistemas complejos
class Proyector {
  encender() { console.log('Proyector: encendido'); }
  apagar()   { console.log('Proyector: apagado');  }
}
class SistemaAudio {
  configurar(volumen) { console.log(`Audio: volumen ${volumen}`); }
  apagar()   { console.log('Audio: apagado'); }
}
class Persianas {
  bajar()  { console.log('Persianas: bajadas'); }
  subir()  { console.log('Persianas: subidas'); }
}
class Reproductor {
  reproducir(pelicula) { console.log(`▶ Reproduciendo: ${pelicula}`); }
  detener() { console.log('⏹ Detenido'); }
}

// Facade
class HomeTheaterFacade {
  constructor() {
    this.proyector  = new Proyector();
    this.audio      = new SistemaAudio();
    this.persianas  = new Persianas();
    this.reproductor = new Reproductor();
  }

  verPelicula(pelicula) {
    this.proyector.encender();
    this.persianas.bajar();
    this.audio.configurar(30);
    this.reproductor.reproducir(pelicula);
  }

  terminar() {
    this.reproductor.detener();
    this.audio.apagar();
    this.persianas.subir();
    this.proyector.apagar();
  }
}

// Cliente solo interactúa con la fachada
const cine = new HomeTheaterFacade();
cine.verPelicula('El Padrino');
cine.terminar();
Java
public class HomeTheaterFacade {
    private Proyector proyector   = new Proyector();
    private SistemaAudio audio    = new SistemaAudio();
    private Persianas persianas   = new Persianas();
    private Reproductor repr      = new Reproductor();

    public void verPelicula(String pelicula) {
        proyector.encender();
        persianas.bajar();
        audio.configurar(30);
        repr.reproducir(pelicula);
    }

    public void terminar() {
        repr.detener();
        audio.apagar();
        persianas.subir();
        proyector.apagar();
    }

    public static void main(String[] args) {
        HomeTheaterFacade cine = new HomeTheaterFacade();
        cine.verPelicula("El Padrino");
        cine.terminar();
    }
}

🚫 Cuándo NO usar

  • Cuando los clientes necesitan acceso granular a las funcionalidades internas del subsistema.
  • Si la fachada se convierte en un objeto 'dios' con demasiadas responsabilidades.
  • En sistemas simples donde la fachada no reduce significativamente la complejidad.
